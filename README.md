## 主要特性

本软件基于串口与下位机进行通信，以接收纸币波形数据。该软件包含以下功能：

- 单张识别模式。配合下位机仅识别单张纸币，并使其时域与频域波形可视化以进行直观的观察与调试。

- 连续识别模式。配合下位机不间断识别纸币，并显示各类纸币数目以及合计数额。

- 存储功能。可见串口接收数据以 .txt 保存到外部存储器中。
